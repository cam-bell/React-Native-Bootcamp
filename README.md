# React-Native-Bootcamp


Welcome to the **React Native Bootcamp**! This bootcamp is designed to help you build mobile applications using React Native and Expo. By the end of this course, you will have a solid understanding of React Native, navigation with Expo Router, Firebase integration, and app deployment.

## 🛠 Prerequisites
Before starting, ensure you have the following:
- Basic knowledge of JavaScript and React.
- A laptop with internet access.
- **Expo Go** app installed on your smartphone.

### Required Installations
- [Node.js](https://nodejs.org/)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- A code editor (e.g., [Visual Studio Code](https://code.visualstudio.com/))
- [Expo Go](https://expo.dev/client) (for running apps on your phone)

---
## 📅 Bootcamp Schedule

### **Day 1: Introduction to React Native & Expo**
**Objective:** Set up the development environment and understand the basics of React Native.

#### Topics Covered:
1. **Introduction to React Native**  
   - What is React Native?
   - Benefits of using React Native.
   - Overview of Expo.
2. **Setting Up the Development Environment**  
   - Installing Node.js and npm.
   - Installing Expo CLI.
   - Setting up a code editor.
3. **Creating Your First React Native App**  
   - Initializing a new Expo project.
   - Understanding the project structure.
   - Running the app on the Expo Go app.
4. **Basic Components and Styling**  
   - Introduction to React Native components.
   - Styling with `StyleSheet`.
   - Creating and styling basic components (`Text`, `View`, `Button`).

✅ **Homework:** Create a simple app with a few styled components.

---
### **Day 2: Navigation with Expo Router**
**Objective:** Implement navigation in a React Native app using Expo Router.

#### Topics Covered:
1. **Introduction to Navigation**  
   - Importance of navigation in mobile apps.
   - Overview of different navigation libraries.
2. **Setting Up Expo Router**  
   - Installing Expo Router.
   - Configuring the router in your project.
3. **Creating Navigation Structure**  
   - Defining routes and screens.
   - Navigating between screens.
   - Passing data between screens.
4. **Advanced Navigation Concepts**  
   - Nested navigation.
   - Tab and drawer navigation.
   - Handling navigation state.

✅ **Homework:** Implement a multi-screen app with navigation using Expo Router.

---
### **Day 3: Advanced Topics, Firebase Integration, and Deployment**
**Objective:** Learn state management, API integration, Firebase setup, and app deployment.

#### Topics Covered:
1. **State Management**  
   - Introduction to state management.
   - Using React's Context API.
   - Introduction to Redux (optional).
2. **Working with APIs**  
   - Fetching data from an API.
   - Displaying data in your app.
   - Handling loading and error states.
3. **Firebase Integration**  
   - Setting up a Firebase project.
   - Installing Firebase SDK.
   - Storing and fetching data from Firebase Firestore.
4. **Deploying the App**  
   - Using EAS Update for over-the-air updates.
   - Overview of deploying to the Play Store & App Store using EAS Submit.
5. **Final Project:** Build a small React Native app integrating the topics covered.

✅ **Homework:** Complete the project and prepare for a demo.

---
## 🎯 Summary
By the end of this bootcamp, you will:
✅ Have hands-on experience with React Native and Expo.
✅ Understand navigation with Expo Router.
✅ Learn how to integrate Firebase for backend services.
✅ Be able to deploy apps to Google Play Store and Apple App Store.
